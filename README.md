# Fifa-Tournament-Generator

How to Use:

Save these three files in the same folder: index.html, styles.css, script.js
Open index.html in your web browser

Setup your tournament:

Enter tournament details
Add player names and teams
Configure games per team and knockout spots

League Phase:

Click "Generate Schedule" to create random matchups
Enter results for each match
View automatic standings updates

Knockout Phase:

Proceed when all league matches are complete
The app automatically creates playoffs if needed
Continue entering results until you have a winner!

Features Included:
✅ Random match scheduling
✅ EPL-style tiebreakers
✅ Playoff system for non-traditional knockout numbers
✅ Data persistence (survives browser refresh)
✅ Visual tables and brackets
✅ Responsive design for all devices

The app is ready to use! Just open the HTML file in a browser and start creating your FIFA tournaments. All data automatically saves locally in your browser.
